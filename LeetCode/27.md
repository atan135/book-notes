## 27. Remove Elements

**题目：**

![](http://p9zl5r4hu.bkt.clouddn.com/2018-10-29leet_27.png)

**解答：**

类似题26，注意下删除后的元素的移动方式即可

**代码：**

```cpp
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        if(nums.empty())    return 0;
        int len = nums.size();
        int lenafter = 0, head = 0;
        for(int i = 0;i < len; ++i){
            if(nums[i] == val)
                ++head;
            else{
                if(i != lenafter){
                    nums[lenafter] = nums[i];
                }
                ++lenafter;
            }
        }
        return lenafter;
    }
};
```
