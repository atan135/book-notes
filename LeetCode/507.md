## 507. Perfect Number

**题目：**

![](http://p9zl5r4hu.bkt.clouddn.com/2018-10-14_59leet_507.png)

**解答：**

这里的题意是，一个数和自己的子因数之和相等，使用类似判定质数的方法，遍历从2到sqrt(n)，如果能够整除，那么这个数a和 n / a 都属于数n的子因数。之后确认即可

只需要区分一下1这个特殊情况

**代码：**

```cpp
class Solution {
public:
    bool checkPerfectNumber(int num) {
        int sum = 0;
        int max = (int)sqrt(num);
        for(int i = 2;i <= max; ++i){
            if(num % i == 0){
                sum += i;
                sum += num / i;
            }
        }
        if(num != 1)
            sum += 1;
        return sum == num;
    }
};
```
