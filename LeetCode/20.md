## 20. Valid Parentheses

**题目：**

![](http://p9zl5r4hu.bkt.clouddn.com/2018-10-25leet_20.png)

**解答：**

这个看代码吧，使用栈记录所有左括号，然后如果遇到有匹配的右括号，就将其pop出来，否则报错

**代码：**

```cpp
class Solution {
public:
    bool isValid(string s) {
        stack<int> stk;
        stk.push(0);
        for( string::iterator it = s.begin();it != s.end();++it )   {
            if( *it == '(' )    
                stk.push(1);
            else if( *it == '[' )
                stk.push(2);
            else if( *it == '{' )
                stk.push(3);
            else if( *it == ')' )   {
                if( stk.top() == 1 )
                    stk.pop();
                else
                    return false;
            }
            else if( *it == ']' )   {
                if( stk.top() == 2 )    
                    stk.pop();
                else 
                    return false;
            }
            else if( *it == '}' )   {
                if( stk.top() == 3 )
                    stk.pop();
                else
                    return false;
            }
        }
        if( stk.top() == 0 )    return true;
        return false;
    }
};
```
